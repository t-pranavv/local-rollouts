class phyagi.optimizers.schedulers.warmup_decay.WarmupDecayCooldownLR(optimizer: Optimizer, total_num_steps: int = 1000, warmup_min_lr: float | List[float] = 0.0, warmup_max_lr: float | List[float] = 0.001, warmup_num_steps: int = 1000, warmup_patience_num_steps: int = 0, warmup_type: str = 'linear', decay_min_lr: float | List[float] | None = None, decay_num_steps: int = 0, decay_patience_num_steps: int = 0, decay_type: str = 'cosine', decay_cycles: float = 0.5, cooldown_min_lr: float | List[float] | None = None, cooldown_num_steps: int = 0, last_batch_iteration: int = -1)[source]#

Learning rate scheduler with warmup, decay and cooldown periods.

This scheduler supports five stages:

  1. Warmup: Linearly increase learning rate from warmup_min_lr to warmup_max_lr.

  2. Patience: Wait for warmup_patience_num_steps steps before decaying.

  3. Decay: Decay learning rate from warmup_max_lr to decay_min_lr.

  4. Patience: Wait for decay_patience_num_steps steps before cooling down.

  5. Linear cooldown: Linearly decrease learning rate from decay_min_lr to cooldown_min_lr.
