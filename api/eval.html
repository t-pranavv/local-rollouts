phyagi.eval.registry.run_task(task_name: str, model: Module, tokenizer: PreTrainedTokenizerBase, device: int | device | None = None, **kwargs) → Dict[str, Any][source]#

Run an evaluation task.

This function wraps the run() method of the task class. It is a convenience function for running a task without having to import the task class directly.

Since each task might have a different set of keyword arguments, this function takes keyword arguments and passes them directly to the run() method. However, the following arguments are required to all tasks: model and tokenizer.

Parameters:

  • task_name â€“ Name of the task.

  • model â€“ Instance of a model.

  • tokenizer â€“ Instance of a tokenizer.

  • device â€“ Device to use for running the task.
